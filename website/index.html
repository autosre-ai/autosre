How It Works

Get started in 3 steps

1

Install

One command to install OpenSRE

pip install opensre
2

Configure

Point to your Prometheus and LLM

export OPENSRE_PROMETHEUS_URL=...
export OPENSRE_LLM_PROVIDER=ollama
3

Investigate

Run your first investigation

opensre investigate "high error rate"
